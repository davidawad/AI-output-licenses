# AI-output-licenses

This repository contains various licenses for AI-generated content. The purpose of this repository is to provide clear template licenses for AI model creators to clarify the desired liability of the outputs of their models.

Feel free to contribute "AI-output-licenses" of other kinds as well.

#### [GPLv3-ai](./GPLv3-ai.txt)

> For a clear example of the specific changes to the original GPL-3.0 license, [take a look at this document here for more clarity](https://docs.google.com/document/d/1EF3-FLKDO0L9WATKeTaya_Ho9RsWW7YGabL-FkTiWag/edit?tab=t.0#heading=h.xc53oz7o84mx).

Note that this license is based on the GNU General Public License v3.0, with specific modifications to address AI-generated content, the idea being that the CONTENT itself should also be publicly available.
